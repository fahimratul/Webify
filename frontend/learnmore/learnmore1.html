How to Use WEBIFY

Learn, step by step, how to go from an empty canvas to a live static website using WEBIFY’s local-first, no-code editor.

Local-only editing Static export ready

Workflow in 6 Simple Steps

  1. Log in to WEBIFY
  2. Create or open a project
  3. Design your page with drag-and-drop blocks
  4. Write and refine your content
  5. Preview and fine‑tune the layout
  6. Export static files and publish

1. Create or Open a Project

From the dashboard, click Create New Project to start a fresh website, or choose an existing project card to continue where you left off.

WEBIFY stores projects directly in your browser, so there is no login and no server-side account to manage.

2. Pick a Template or Start Blank

Choose a starter template such as a portfolio, resume, or landing page if you want a ready-made layout to tweak.

You can also start from a blank canvas for full control and build every section yourself using layout blocks.

3. Design with Drag-and-Drop

Use the visual editor to drag text, images, buttons, headers, footers, and containers onto your canvas, then move and resize them to shape your page.

Styling options let you adjust colors, fonts, spacing, and alignment without writing any HTML or CSS.

4. Write Content (Optionally with AI Help)

Click on any text block to edit headings, paragraphs, and buttons directly on the page, keeping everything in context.

If enabled, the local LLM assistant can suggest copy, section ideas, or layout improvements while keeping all data on your device.

5. Preview Your Site

Use the preview mode to see the site without editor controls and quickly check how it feels as a real web page.

Switch between desktop and mobile breakpoints to make sure your layout and typography look good on different screen sizes. [web:4]

6. Export Static Files

When you are satisfied, click Export to download a bundle of static HTML, CSS, and JavaScript files that contain your site.

These files can be uploaded to any static hosting provider such as GitHub Pages or Netlify, making your site fast, secure, and easy to host. [web:10]

Tips for Effective Use

  • Work in small sections: finish your hero, then portfolio, then contact.
  • Reuse blocks and styles to keep pages visually consistent.
  • Preview often and adjust spacing for clean, breathable layouts.

Common Questions

  • Do I need the internet? No, editing works offline; you only need it to download WEBIFY and upload your final site.
  • Where is my data stored? All projects are stored locally in your browser storage until you export them.
  • Can I edit the code? Yes, exported files are plain static assets you can open and tweak in any code editor. [web:4][web:10]
